# ecommerce-sales-analysis# E-commerce Sales Analysis & Forecasting

Analyzed 50K+ sales transactions to identify customer purchase trends and predict revenue.

## Tools Used
- Python, SQL, Power BI, Pandas, Scikit-learn

## What I Did
- Customer segmentation and churn prediction using ML
- Built interactive Power BI dashboards for KPI tracking
- Data cleaning and transformation using Pandas
- SQL queries for aggregation and trend analysis
